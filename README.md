 Workout Forge — Дневник тренировок на 7 языках
Workout Forge — коллекция из семи независимых реализаций дневника тренировок, каждая из которых работает на своём языке программирования. Проект создан для демонстрации кроссплатформенного подхода и предоставления удобного инструмента для отслеживания физической активности, прогресса и достижений.

✨ Уникальные возможности (общие для всех версий)
📝 Запись тренировок: упражнение, подходы, повторения, вес, дата.

📊 Статистика:

Общее количество тренировок.

Общий объём (сумма повторений × вес).

Прогресс по каждому упражнению (график в CLI/GUI).

🔍 Фильтрация по упражнению, дате, диапазону веса.

📈 Визуализация прогресса (ASCII-график в консоли, графики в GUI).

💾 Сохранение в JSON-файл.

📤 Экспорт в CSV для анализа в Excel.

🏆 Личные рекорды — лучший подход для каждого упражнения.

🖥️ Интерфейсы:

Командная строка (CLI) с интерактивным меню.

Графический интерфейс (GUI) в Python, Java, C#.

Веб-интерфейс в PHP.

🚀 Быстрый старт
Установка зависимостей (при необходимости)
Python
bash
pip install tkinter  # (опционально для GUI)
JavaScript (Node.js)
bash
npm install commander
Go
bash
go get -u
Rust
bash
cargo add clap serde_json chrono
Java
Не требует внешних библиотек (используется стандартный java и javax.swing).

C#
Не требует внешних библиотек (используется System.Windows.Forms и System.Text.Json).

PHP
Не требует внешних библиотек.

Использование (общий синтаксис)
CLI (все версии):

bash
workout add --exercise "Приседания" --sets 3 --reps 10 --weight 50
workout list --exercise "Приседания"
workout stats
workout pr
Интерактивный режим (CLI):
Запустите без аргументов – откроется меню.
